* 향상된 __Binding__
  * `:property_name` 포멧의 바인딩을 지원합니다.
  

  ```cs
  class UserProfile : APIBase {
     string username {get;set;}
  
     [Resource(":username/profile")]
     string GetProfile();
  }
  ```
  
* 인터페이스에서의 `Task<>` 반환형 지원

  ```cs
  [Resource(":username/profile")]
  Task<string> GetProfile();
  ```
